## 水平垂直居中

以div为例

```css
main {
    height: 100vh;
    display: flex;
    justify-content: center;
    align-items: center;
}
```

效果图：

![CSS-layout_2019-09-04_23-52-22.png](https://i.loli.net/2019/09/04/wKd6jRsoEykQfIv.png)





## 同行等比例布局

```css
display: flex;
align-items: flex-start;
```

## 垂直居中布局

```css
display: flex;
flex-direction: column;
justify-content: center;
```
